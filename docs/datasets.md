# Planned Datasets (for Milestone 2 — Pose Estimation)

- **Human3.6M** — large-scale human pose estimation dataset with 3D joint tracking, useful for training baseline pose models.
- **MPII Human Pose** — dataset for 2D body keypoint detection across diverse everyday activities.
- **COCO Keypoints** — widely-used benchmark dataset for pose estimation training and evaluation.
- **SportsPose** — sports-specific movement and pose dataset, directly relevant to athletic motion analysis.
- **FIFA Injury Dataset** — reference dataset for injury trends and risk factors in football/sports contexts.

These datasets are identified for integration in Milestone 2. No video data has been downloaded yet — this milestone only requires documentation of intended sources.
